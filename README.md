## Specifications

- Shell: `zsh`
- Terminal: `kitty`
- Text editor: `nvim` [`vim`]
- Browser: `firefox` [`brave-browser`]
- Compositor: `picom`
- File manager: `ranger` [`thunar`, `nemo`]
- PDF reader: `zathura`
